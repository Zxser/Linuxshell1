仓库脚本的使用方法：

1、对于脚本文件：

打开需要下载的shell脚本，RAW视图模式，直接wget下载即可，也可以用curl -s 命令  

第二种方法就是通过httpdie工具来下载，前提是如果你对wget 或者curl -s不熟悉

2、对于压缩文件

打开需要下载的包，点击View Raw，即可下载  复制链接地址即可，工具可以使用wget 或者是curl -s
